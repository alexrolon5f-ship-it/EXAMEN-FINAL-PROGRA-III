# TravelPy

Trabajo para examen de Programación III. Sitio web de una agencia de viajes, hecho con HTML5, CSS3 y JavaScript.

Autor: Alex Rolon 
## Estructura de la agencia de viaje

- `index.html` - Inicio
- `nosotros.html` - Historia, misión, visión, valores y equipo
- `servicios.html` - Paquetes de viaje
- `galeria.html` - Galería de imágenes
- `blog.html` - Noticias
- `faq.html` - Preguntas frecuentes
- `contacto.html` - Formulario de contacto (EmailJS) y ubicación
- `desarrolladores.html` - Inforomacion del desarrollador
- `privacidad.html` - Política de privacidad
- `CSS/` - hojas de estilo
- `JS/` - scripts (modo oscuro, modales, validación de formulario)
- `IMG/` - imágenes propias del sitio
- `fonts/` - tipografías usadas (Montserrat, ZenDots)
